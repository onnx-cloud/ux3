## How To Use Examples

1. Start from the closest example structure.
2. Copy the folder patterns, not app-specific business logic.
3. Keep widget FSMs, templates, i18n, services, and styles aligned.
4. Run build/test before and after each feature slice.

For IAM-specific notes, see `examples/iam/README.md`.
