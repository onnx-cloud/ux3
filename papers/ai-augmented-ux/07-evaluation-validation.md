This chapter defines evaluation and validation criteria for model-assisted synthesis.

Acceptance and trust are measured by review speed, modification rates, provenance quality, and developer confidence in suggestions. This reflects whether assistance is genuinely useful rather than merely novel.

The evaluation should also compare local deterministic synthesis with cloud-assisted suggestions and measure how well each approach exploits live host context such as session mode, plan state, recent tool outputs, and FSM snapshots.

Case studies should evaluate complete workflows in running applications. The selected cases should document compile-time verification, the fidelity of generated artifacts, the clarity of replayable reasoning traces, and how human steering changed the final outcome.
