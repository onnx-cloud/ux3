# Research Papers

This collection presents a set of formal research papers aligned with a compile-first, plugin-extensible frontend platform. The papers treat the system not only as a compiler, but also as a live host: a runtime that exposes tools, prompts, resources, session state, replay, inspection, and knowledge retrieval as first-class collaboration surfaces. Across the corpus, that exposed layer is treated as a typed cognitive surface rather than an ad hoc plugin boundary.

Each paper is rated for:

- **Novelty** — relative originality in frontend systems, human-AI interaction, and agentic tooling.
- **Impact** — potential effect on developer workflows, runtime reliability, and organizational learning.
- **Innovation** — strength of the architectural, semantic, or interaction contribution.
- **Research grade** — suitability for academic or R&D publication.

## Overlooked Supported Innovations

- **Typed cognitive surface** — tools, prompts, and resources are exposed as structured runtime contracts, not ad hoc extensions.
- **Stateful agent sessions** — human and agent workflows already support blocking, queueing, and steering modes rather than one-shot prompting.
- **Host memory and self-observation** — dev-session state, FSM snapshots, and invocation history create a queryable operational memory.
- **Replay and inspection** — event capture, session replay, and the live inspector make the system self-documenting and experimentally observable.
- **Deterministic knowledge retrieval** — build-time indexing and runtime-accessible knowledge resources reduce dependence on opaque retrieval paths.
- **Executable semantics** — semantic math, FSM artifacts, validation rules, and provenance metadata form a basis for lightweight formal methods.
