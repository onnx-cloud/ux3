UX3 Framework Demo: Todo App + CDN Script Loading

Demo Instructions:

1. Login View: Shows login form (visible in auth.idle/error)
2. Todo List View: Shows todos (visible in todoList.loaded)
3. CDN Scripts View: Demonstrates ux-repeat on script tags
   - Click "Load CDN Scripts" to trigger ux-repeat
   - Scripts are created in the hidden container
   - Each script gets correct src attribute via reconcileScriptRepeat()
   - This proves ux-repeat works for dynamic script injection

Key fix demonstrated:
- Old way: <script ux-repeat="site.cdn" src=""></script>
  Problem: src attribute not set, script tag not executed
  
- New way: reconcileScriptRepeat() in _render()
  Solution: Properly sets src, creates actual script elements
  Result: Scripts are loaded from CDN correctly
