# Home - Getting Started

Welcome to Kanban! This is your project hub where you can manage your work boards and stay organized.

## What You Can Do

### View Your Projects
- See all your active projects at a glance
- Each project card shows key statistics and recent activity
- Click any project to open its boards

### Create a New Project
- Click the **"+ New Project"** button to get started
- Set a project name and optional description
- Choose a project template (if available) or start blank

### Manage Your Workspace
- **Archive projects** you've completed
- **Restore archived** projects from history
- **Delete projects** permanently (if you have permission)

## Tips & Tricks

💡 **Keyboard Shortcut**: Press `N` to quickly create a new project

💡 **Search**: Use the search bar to find projects by name

💡 **Starred Projects**: Click the star icon to pin your favorite projects to the top
