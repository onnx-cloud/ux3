# Board List - Select a Board

This view shows all the boards in your project. Boards are groupings of related tasks organized into lanes.

## What You Can Do

### View All Boards
- See every board in your project
- Each board displays its name, description, and lane count
- Boards are color-coded by status (active, archived, etc.)

### Create a New Board
- Click **"+ New Board"** to create a fresh workspace
- Give it a meaningful name (e.g., "Sprint 24", "Bug Fixes")
- Add an optional description to explain its purpose

### Manage Boards
- **Edit** a board's name or description
- **Duplicate** a board to reuse its lane structure
- **Archive** a completed board
- **Delete** a board (moves to trash first)

## Board Types

- **Sprint**: Time-boxed boards for iterative work
- **Backlog**: Long-term task storage
- **Process**: Standard workflow (To Do → In Progress → Done)
- **Custom**: Your own lane structure

## Quick Actions

- **Hover** over a board to see action buttons
- **Right-click** for context menu options
- **Double-click** to rename quickly

## Tips

💡 Use consistent board naming for easier navigation

💡 Archive completed boards to keep your workspace clean

💡 Duplicate successful board templates to save setup time
